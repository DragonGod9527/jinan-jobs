## 📖 关于本站

**济南就业参考** 是一个收录济南本地公司真实评价的社区平台，帮助求职者在入职前了解公司真实情况，少走弯路，避免踩坑。

---

## 🕯️ 关于原站

原「济南就业参考」网站（gshmd.vip）因**不可抗力原因**，于 **2026年2月13日** 永久关闭。

在关站前，原站长慷慨地将所有帖子数据开放备份，本站基于这份数据重建，旨在让这些珍贵的真实声音得以延续。

**感谢原站长的付出与坚持。** 🙏

## 🚀 功能特点

- 🔍 **快速搜索** - 输入公司名，秒查评价
- 🔥 **热度排序** - 查看最受关注的公司
- 🕐 **时间排序** - 浏览最新发布的评价
- 📱 **响应式设计** - 手机电脑都能用
- ✏️ **开放发帖** - 任何人都可以分享经历

---

## 📝 如何发布评价

1. 点击网站右下角的 ✏️ 按钮
2. 使用 GitHub 账号登录
3. 按照模板填写公司信息和你的评价
4. 提交即可

**匿名保护**：GitHub Issue 不会显示你的真实姓名，只显示 GitHub 用户名。
