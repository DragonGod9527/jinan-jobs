## 📚 使用说明

### 🔍 如何查询公司？

1. 打开网站：https://dragongod9527.github.io/jinan-jobs/
2. 在搜索框输入**公司名称**（支持模糊搜索）
3. 点击搜索或按回车
4. 点击任意帖子卡片查看详细评价和评论

### 📋 如何浏览？

| 标签 | 说明 |
|------|------|
| 🔥 最热 | 按浏览量排序，查看最受关注的公司 |
| 🕐 最新 | 按时间排序，查看最新发布的评价 |
| 🏢 公司列表 | 浏览所有收录的公司 |

### 💬 如何回复/评论？

1. 点击帖子打开详情
2. 滚动到底部看到评论区
3. 点击 **使用 GitHub 登录**
4. 登录后直接在输入框写评论，提交即可

> 💡 评论会显示在该帖子下方，其他用户可以看到并回复你。

### ✏️ 如何发布新公司评价？

1. 点击网站**右下角的 ✏️ 按钮**
2. 跳转到 GitHub，使用 GitHub 账号登录
3. 按照模板填写：公司名称、地址、你的评价
4. 点击 **Submit new issue** 提交

> 💡 **提示**：没有 GitHub 账号？[点击这里免费注册](https://github.com/signup)，只需邮箱即可。

### 🔒 隐私安全吗？

- ✅ 发帖只显示 GitHub 用户名，不会暴露真实身份
- ✅ 建议使用匿名的 GitHub 账号发布敏感内容
- ✅ 所有数据托管在 GitHub，安全可靠

## 🕯️ 关于原站

原「济南就业参考」网站（gshmd.vip）因**不可抗力原因**，于 **2026年2月13日** 永久关闭。

在关站前，原站长慷慨地将所有帖子数据开放备份，本站基于这份数据重建，旨在让这些珍贵的真实声音得以延续。

**感谢原站长的付出与坚持。** 🙏
